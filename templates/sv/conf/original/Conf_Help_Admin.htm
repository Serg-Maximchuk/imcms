Administrera forum

Lägg till forum: Ange det namn du vill att det nya forumet skall heta. Klicka sedan på knappen "Lägg till ett forum" för att lägga till forumet i konferensen. Observera listan med befintliga forum.

Radera forum: För att radera ett forum i en konferens, välj det forum du vill ta bort och klicka på knappen "Ta bort". Valt forum och alla dess diskussioner kommer då att raderas. Observera att en konferens måste minst ha ett forum för att kunna fungera!

Ändra namn på ett befintligt forum: Markera det befintliga forum som du vill byta namn på i listan. Ange det nya namnet forumet skall anta. Klicka på knappen "Ändra" för att ändra namnet.

Ändra antal diskussioner att visa för ett visst forum: Markera det forum du vill ändra antalet diskussioner som skall visas. Det befintliga antalet diskussioner som visas i forumet anges inom parenteserna. Välj sedan det nya antalet diskussioner som skall visas i diskussionen och klicka sedan på knappen "Uppdatera" för att ändra det antal diskussioner som skall visas samtidigt i forumet.


Administrera diskussioner

Ta bort en diskussion: Klicka i checkboxen framför den diskussion du vill ta bort och klicka sedan på knappen "Ta bort ". Diskussionen och alla kommentarer tillhörande denna kommer att tas bort.

Återgå till användarläge: Klicka på bilden "Avsluta admin".

Administrera kommentarer

Ta bort en kommentar: För att ta bort en kommentar så klicka i checkboxen vid varje inläggsrubrik och klicka på knappen "Ta bort". Inlägget kommer då att tas bort ifrån diskussionen. Viktigt! Observera att en diskussions första inlägg, dvs det som initierade diskussionen inte kan tas bort! För att ta bort första inlägget så får man antingen editera inlägget eller ta bort diskussionen.

Spara om en kommentar: För att editera ett befintligt inlägg, ändra den text som skall ändras och markera inlägget genom att klicka i checkboxen. Klicka på knappen "Spara om" för att spara om de editerade inläggen.

Administrera mallar

De mallfiler som används för en konferens utseende kan ändras. Notera att ändringar inte bör göras på mallsetet "Original"! Skapa istället ett nytt mallset och applicera ändringarna på detta istället.

Övrigt. När en konferens skapas, används som standard ett mallset som kallas för "Original". Uppdatera därför aldrig det mallsetet som kallas för "Original"!

Registrera nytt mallset: För att skapa ett nytt mallset, ange namnet på det nya mallsetet och tryck på "Nytt mallset". Observera att mallsetets namn måste vara ett giltigt biblioteksnamn. Tillämpa därför 8.3 regeln vad beträffar namnet på mallsetet. Detta innebär att inga specialtecken får förekomma i mallsets namnet såsom åäö eller liknande för att vara på den säkra sidan. När ett nytt mallset skapas, kommer mallsetet "Original":s mallfiler att kopieras till det nya mallsetet.


Ändra mallset för en konferens: Markera det mallset du vill skall användas för den konferens du administrerar. Klicka sedan på knappen "Byta mallset" för att byta mallset. Det mallset som används för konferensen presenteras i fetstil i dialogen.

Uppdatera mallfil: För att uppdatera en mallfil till ett befintligt mallset, välj först vilket set som skall uppdateras, och välj sedan vad för typ av mall det är som skall uppdateras (Konferensen stöder 2 typer av filer( bilder eller htmlfiler). När dessa val har gjorts klicka på knappen "Administrera".

Vid nästföljande dialogruta hämtas den fil som skall laddas upp och tryck därefter på knappen "Ladda upp". Då kommer den mallfil som valts att kopieras till servern. Observera att den befintliga mallfilen kommer att skrivas över!

Att skapa egna mallfiler: För att skapa egna mallsidor som skall användas i konferensen, gör så här:

1. Skapa egna versioner av de mallfiler du vill ändra utseendet på. Följande mallar/bilder skall skapas. Eftersom det för varje nytt mallset redan existerar mallfiler, så innebär detta att man inte behöver skapa samtliga mallar om man inte vill. Administrera användare

Administrera användare: En användare som är expert i en konferens kan märkas ut, så att de övriga konferensdeltagarna kan läsa vad en expert har skrivit. En sådan användares inlägg märks upp med symbolen . För att låta en användare bli expert så klicka i adminläget på "Avancerad administration" och välj därefter "Administrera användare". Markera den användare du vill ändra status på och klicka på "Ändra". Klicka sedan i checkboxrutan för att ändra expertstatusen för användaren. Spara den nya statusen genom att klicka på "Spara".
