Administrera forum

Lägg till forum: Ange det namn du vill att det nya forumet skall heta. Klicka sedan på knappen "Lägg till ett forum" för att lägga till forumet i konferensen. Observera listan med befintliga forum.

Radera forum: För att radera ett forum i en konferens, välj det forum du vill ta bort i nedrullningslistan och klicka på knappen "Radera". Valt forum och alla dess diskussioner kommer då att raderas. Observera att en konferens minst måste ha ett forum för att kunna fungera!

Ändra namn på ett befintligt forum: Markera det befintliga forum som du vill byta namn på i listan för befintliga forum. Ange det nya namnet forumet skall ha. Klicka på knappen "Ändra" för att ändra namnet.

Ändra antal diskussioner som visas: Här anger man hur många diskussioner som skall visas åt gången i ett forum. Markera det forum du vill ändra antalet diskussioner i. Det befintliga antalet diskussioner som visas i forumet anges inom parenteserna i listan över befintliga forum. Välj sedan det nya antalet diskussioner som skall visas i diskussionen och klicka på knappen "Uppdatera" för att ändra det antal diskussioner som skall visas samtidigt i forumet.

För att åter gå till användarläge, klicka på bilden "Avsluta admin".

Administrera diskussioner

Ta bort en diskussion: Klicka i checkboxen framför den diskussion ska tas bort och klicka sedan på knappen "Ta bort". Diskussionen och alla kommentarer tillhörande denna kommer att tas bort.

För att åter gå till användarläge så klicka på bilden "Avsluta admin".

Administrera kommentarer

Ta bort en kommentar: För att ta bort en kommentar så klicka i checkboxen vid varje inläggsrubrik och klicka på knappen "Ta bort". En varningsbild visas där du får bekräfta att du verkligen vill ta bort inlägget. Inlägget kommer då att tas bort från diskussionen. OBS att diskussionens första inlägg, dvs., det som initierade diskussionen inte kan tas bort. Det går dock att redigera inlägget alternativt ta bort hela diskussionen. Var då medveten om att samtliga inlägg i diskussionen försvinner.

Spara om en kommentar: För att ändra ett befintligt inlägg, ändra den text som skall uppdateras och markera inlägget genom att klicka i checkboxen. Klicka på knappen "Spara om".

Administrera mallar

De mallfiler som används för en konferens utseende kan ändras. Notera att ändringar inte bör göras på mallsetet "Original" som är standardmallsetet som används när en konferens skapas. Uppdatera därför aldrig det mallsetet som kallas för "Original" utan skapa istället ett nytt mallset och gör ändringarna i det.

Registrera nytt mallset: För att skapa ett nytt mallset, ange namnet på det nya mallsetet och tryck på "Nytt mallset". Observera att mallsetets namn måste vara ett giltigt biblioteksnamn. OBS att mallsetets namn inte får innehålla några specialtecken som åäö. När ett nytt mallset skapas, kommer mallsetet "Original":s mallfiler att kopieras till det nya mallsetet.

Ändra mallset för en konferens: Markera det mallset du vill skall användas för den konferens du administrerar. Klicka sedan på knappen "Byta mallset" för att byta mallset. Det mallset som används för konferensen presenteras i fetstil i dialogen.

Uppdatera mallfil: För att uppdatera en mallfil till ett befintligt mallset, välj först vilket set som skall uppdateras, och välj sedan vad för typ av mall det är som skall uppdateras (Konferensen stöder 2 typer av filer (bilder eller HTML-filer). När dessa val har gjorts klicka på knappen "Administrera".

Vid nästföljande dialogruta hämtas den fil som skall laddas upp genom att trycka på knappen "Browse", sedan letar man reda på filen på sin hårddisk, därefter trycker man på knappen "Ladda upp". Då kommer den mallfil som valts att kopieras till servern. Observera att den befintliga mallfilen kommer att skrivas över!

Att skapa egna mallfiler: För att skapa egna mallsidor som skall användas i konferensen, gör så här:

1. Skapa egna versioner av de mallfiler du vill ändra utseendet på. Följande mallar/bilder skall skapas. Administrera användare

En användare som är expert i en konferens kan märkas ut, så att de övriga konferensdeltagarna kan läsa vad en expert har skrivit. En sådan användares inlägg märks upp med symbolen . För att låta en användare bli expert så klicka i adminläget på "Avancerad administration" och välj därefter "Administrera användare". Markera den användare du vill ändra status på och klicka på "Ändra". Klicka sedan i checkboxrutan för att ändra expertstatusen för användaren. Spara den nya statusen genom att klicka på "Spara".

Administrera självregistrering

För att administrera självregistrerade användare klickar man på länken "Självregistrering". Sidan används för att tilldela självregistrerade användare i konferensen en viss roll. När användaren registrerar sig tilldelas han/hon den/de roller som står i rutan under Befintliga. Lägga till en roll: markera den i den vänstra rutan, klicka sedan på "-->". Rollen flyttas över i det högra rutan och nya självregistrerade användare kommer att tilldelas den rollen (och övriga också om det finns fler i rutan). Ta bort en roll: markera rollen i den högra rutan och klicka på "<--". Rollen flyttas över till det vänstra fönstret och nya självregistrerade användare kommer inte att tilldelas den rollen.
