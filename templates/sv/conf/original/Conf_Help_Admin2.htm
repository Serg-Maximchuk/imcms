Hjälpsida för konferensadministratörer

För att skapa nya htmlmallar finns det ett antal taggar per mallsida som man kan använda sig av.
Några taggar kan alltid användas i samtliga mallsidor. Dessa följer nedan. Efter dessa följer de taggar respektive
de kommando som kan skickas till de olika servlets som används i konferensen.

Gemensamma taggar.

Tag Betydelse Exempel
#SERVLET_URL# Infogar sökvägen till servletmappen. #SERVLET_URL#
#IMAGE_URL# Infogar sökväg till konferensen bildmapp #IMAGE_URL#

Gemensamma parametrar
En del parametrar använder sig servletarna av från och till. Dessa hämtas i första hand från requestobjektet och om det inte har angetts där hämtas det från sessionobjektet.

Parametrar Betydelse
META_ID Anger meta id: et för konferensen
PARENT_META_ID Anger vilket meta_id som "vårt" meta_id är löv under
COOKIE_ID Oanvänt i denna modul.
FORUM_ID Anger vilket forum som är aktuellt
DISC_ID Anger vilken diskussion som är aktuell
LAST_LOGIN_DATE Anger vilket datum användaren sist var inloggad i konferensen
USER_ID Anger vilken användare som är inloggad
DISC_INDEX Anger vilket värde som diskussionsindex har. Dvs vilken diskussion som används för att beräkna vilka diskussioner som skall visas för användaren.

 

ConfAdd servlet
Post parametrar: ConfAdd servlet stöder följande parametrar.

Parametrar Betydelse Värden
CANCEL Anger att användaren vill avbryta skapandet av inlägg/diskussion Godtyckligt
ADD Anger att användaren vill lägga till nytt diskussion/inlägg Godtyckligt

Add parametrar: Om funktionsparametern är add krävs följande parametrar:

ADD_HEADER Anger rubriken på inlägget Godtycklig
ADD_TEXT Anger text på inlägget Godtycklig
ADDTYPE Anger vilken typ av inlägg som görs. Kommentar eller starta diskussion REPLY, DISCUSSION

Tag Betydelse

#DISC_HEADER# Infogar aktuell diskussions rubrik

ConfAdmin servlet
Taggar: Följande taggar stöds.

Tag Betydelse Get parameter
#EXTERNAL_PATH# Används i adminläge för att ange var någonstans en uppladdad fil skall placeras på server. META
#TEMPLATE_LIST# Anger en lista med vilka mallset som finns META
#A_META_ID# Anger vilket meta id den aktiva konferensen har META
#CURRENT_TEMPLATE_SET# Används för att visa vilket templateset som används META
#FORUM_LIST# Anger en lista med de forum som finns i konferensen FORUM
#NBR_OF_DISCS_TO_SHOW_LIST# Anger en speciell form av forumulista som förutom forumnamnen även innehåller antalet diskussioner som visas FORUM
#NEW_A_HREF_LIST# Infogar alla nya diskussioner sedan admin sist var inloggad DISCUSSION
#OLD_A_HREF_LIST# Infogar alla gamla diskussioner sedan admin sist var inloggad DISCUSSION
#USER_SORT_ORDER# Infogar användarens sorteringsorder på inläggen. Kan vara 1 eller 0. REPLY
#CHECKBOX_STATE# Infogar ordet checked / eller inget beroende på användarens sorteringsorder. REPLY
#REPLIES_RECORDS# Infogar lista med samtliga inlägg i en diskussion REPLY
#CONF_UNADMIN_BUTTON# Infogar bild på knapp för avsluta administration
#CONF_UNADMIN_LINK# Infogar knapp för att avsluta administration

Postparametrar:

Parameter Betydelse parameter värde
REGISTER_TEMPLATE_LIB Registrerar ett nytt mallset godtyckligt
UPLOAD_CONF Anger namnet på det första forumet godtyckligt
SET_TEMPLATE_LIB Anger vilket mallset som skall användas i en diskussion godtyckligt
DELETE_REPLY Tar bort ett inlägg i en diskussion godtyckligt
RESAVE_REPLY Sparar om ett editerat inlägg godtyckligt
DELETE_DISCUSSION Tar bort en diskussion godtyckligt
DELETE_FORUM Tar bort ett forum godtyckligt
ADD_FORUM Lägger till ett forum godtyckligt
CHANGE_FORUM_NAME Ändrar namnet på ett befintligt forum godtyckligt
SHOW_DISCUSSION_NBR Sätter antalet diskusssioner som skall visas i ett forum godtyckligt

REGISTER_TEMPLATE_LIB parametrar: Om funktionsparametern är REGISTER_TEMPLATE_LIB krävs följande parametrar.

TEMPLATE_LIB_NAME Anger det nya namnet på det nya mallsetet Godtycklig

UPLOAD_CONF parametrar: Om funktionsparametern är UPLOAD_CONF krävs följande parametrar.

TEMPLATE_NAME Anger namnet på det mallset uploaden skall utföras på Befintligt mallset (väljs ur lista)
UPLOAD_TYPE Anger filtypen som laddas upp. IMAGE/TEMPLATE

SET_TEMPLATE_LIB parametrar: Om funktionsparametern är SET_TEMPLATE_LIB krävs följande parametrar.

TEMPLATE_ID Anger id:et på det nya templatesetet som väljs Befintligt mallset (väljs ur lista)

RESAVE_REPLY parametrar: Om funktionsparametern är RESAVE_REPLY krävs följande parametrar.

TEXT_BOX_x Anger Texbox id:et på ett inlägg. Exempel: TEXT_BOX_2 // Anger textbox nr 2
REPLY_HEADER_x Anger Texbox id:et på ett inlägg. Exempel: REPLY_HEADER_2 // Anger inläggsrubrik nr 2

ConfDisc servlet
Taggar: Följande taggar stöds.

Tag Betydelse
#A_HREF_LIST# Infogar lista med samtliga diskussioner i ett forum. Kommer i en a href tag
#PREVIOUS_BUTTON# Infogar knapp för lista förgånede diskussioner.
#NEXT_BUTTON# Infogar knapp för lista näst diskussioner.
#NEW_DISC_BUTTON# Infogar knapp för att lägga till diskussioner.
#CONF_ADMIN_BUTTON# Infogar bild på administrationsknapp
#CONF_ADMIN_LINK# Infogar knapp för administration av diskussion

Getparametrar: Inga
Postparametrar:

Parameter Händelse parameter värde
UPDATE Uppdaterar diskussionsservleten godtyckligt
NEXT Generar en ny discussionslista med x antal senare skapade diskussioner om det finns sådana. godtyckligt
PREVIOUS Generar en ny discussionslista med x antal tidigare skapade diskussioner om det finns sådana. godtyckligt
SEARCH Generar en ny discussionslista med alla träffar som gjorts för en sökning godtyckligt

Sökparametrar:
Följande parametrar kommer att behövas om parametern SEARCH har skickats till ConfDisc Post funktion.

Sök parameter Innebörd parameter värde
CATEGORY Anger vilken kategori sökningen skall utföras i. 0=rubriker, 1=texter, 2=användare 0/1/2
SEARCH Anger sökordet/orden som skall sökas efter godtyckligt
FR_DATE Anger startdatum som sökningen skall gälla för godtyckligt
FR_VALUE Anger det defaultvärde som står i datumrutan. För svenska gäller Igår. FR_VALUE skall överenstämma m FR_DATE defaultvärde godtyckligt
TO_DATE Anger slutdatum som sökningen skall gälla för godtyckligt
TO_VALUE Anger det defaultvärde som står i datumrutan. För svenska gäller Idag. TO_VALUE skall överenstämma m TO_DATE defaultvärde godtyckligt
