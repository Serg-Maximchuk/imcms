Administration page for the browser control document.

Regarding the buttons below:

  • Normal allows you to go through the BC-document to the page defined for your browser.
  • Edit allows you to edit the BC-document.
  • Back takes you back to the last textdocument visited.

The other buttons work as usual.

NB! If you wish to return to this administration page later on, click on New links and then on the small red arrow by the link to this document.
