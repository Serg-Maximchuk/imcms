Att ändra avdelning

Varje anslagstavla har minst en avdelning, där olika anslag visas. Välj vilken avdelning du vill gå in i och klicka på "Välj".


Att läsa anslag

Det här är rubriken 03-20 16:59 [4]

De anslag som visas i fönstrets vänstra del är rubrikerna till de senaste anslagen som skrivits i respektive avdelning.

Genom att klicka på anslagssrubriken visas hela anslaget till höger i fönstret.


Att skapa anslag/ svar
Ett anslag skapas genom att man klickar på "Nytt Anslag", skriver ett inlägg i inmatningsrutan och avslutar med att trycka på knappen "Skicka". Använd knappen "Ångra" för att avbryta utan att spara anslaget.
Ett svar skapas genom att man klickar på "Svara", skriver sitt inlägg i inmatningsrutan och avslutar med att trycka på knappen "Skicka". Använd knappen "Ångra" för att avbryta utan att skicka svaret.

Övrigt: Mellan rubrikfält och textfält finns en länk, "LÄNK". När man trycker på den visas en dialogruta där man anger den webadress/mailadress som skall ingå i textfältet. När man trycker på "OK" skapas automatiskt den html kod som behövs för att infoga länken. Använd knappen "Ångra" för att avbryta utan att infoga länken.

Att uppdatera anslag
För att visa de anslag som tillkommit sedan du loggade in används knappen "Uppdatera".

Att bläddra bland anslag
 De anslag som visas i AVDELNINGSrutan är de senast tillkomna. För att leta bland äldre diskussioner används knapparna "Senare" och "Tidigare" som ger dig möjlighet att bläddra bland alla anslag.


Att uppdatera anslag
För att visa de anslag som tillkommit sedan du loggade in används knappen "Uppdatera".

Att söka i anslagstavlan
Sökning kan göras bland texternas rubriker, innehåll och författare inom en viss avdelning. Ange sökord och tryck på knappen "Sök". Sökningen kan begränsas genom att ange olika start och sludatum. Ett datum anges i formen årtal-månad-dag (YYYY-MM-DD), eller i specialformerna "igår" och "idag".
Exempel på sökdatum är: 2000-04-02
