Att ändra avdelning

Varje anslagstavlahar minst en avdelning, där olika anslag visas. Välj vilket avdelning du vill gå in i och klicka på Ok.


Att läsa anslag

Det här är rubriken 03-20 16:59 [4]

De anslag som visas i fönstrets vänstra del är rubrikerna till de sista anslagen som skrivits i respektive avdelning.

Genom att klicka på anslagssrubriken visas hela anslaget till höger i fönstret.


Att skapa anslag/ svar
Ett anslag skapas genom att man klickar på "NYTT ANSLAG", skriver ett inlägg i inmatningsrutan och avslutar med att trycka på knappen "Spara". Använd knappen "Avbryt" för att avbryta utan att spara anslaget.
Ett svar skapas genom att man klickar på "svara", skriver sitt inlägg i inmatningsrutan och avslutar med att trycka på knappen "Skicka". Använd knappen "Avbryt" för att avbryta utan att skicka svaret.

Övrigt: Mellan rubrikfält och textfält finns två länkar, "LÄNKAR" resp. "MAIL". När man trycker på någon av dessa visas en dialogruta där man anger den webadress/mailadress som skall ingå i textfältet. När man trycker på "Ok" skapas automatiskt den html kod som behövs för att infoga länken.

Att bläddra bland diskussioner
 De diskussioner som visas i diskussionsrutan är de senast tillkomna. För att leta bland äldre diskussioner används knapparna "Senare" och "Tidigare" som ger dig möjlighet att bläddra bland alla diskussioner.


Att uppdatera inlägg
För att visa de inlägg som tillkommit sedan du loggade in används knappen "Uppdatera".

Att söka i konferensen
Sökning kan göras bland texternas rubriker, innehåll och författare inom ett visst forum. Ange sökord och tryck på knappen "Sök". Sökningen kan begränsas genom att ange olika start och sludatum. Ett datum anges i formen årtal-månad-dag (YYYY-MM-DD), eller i specialformerna "igår" och "idag".
Exempel på sökdatum är: 2000-04-02
