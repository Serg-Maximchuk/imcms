Administrate forum

Create a new forum: Enter a name for the forum to be created. Then click on the button "Create new forum" to add the new forum within the conference. Please note the list of existing forums.

Delete a forum: To delete a forum in a conference, select that forum you wish to remove and then click on the button "Delete". The selected forum and all of its discussions will then be deleted. Please note that a conference must have at least one forum to be able to funtion!

Change the name of an existing forum: Select the existing forum whose name you wish to change from the list of forums. Enter the new name of the forum. Click the button "Change" to let the new name replace the existing name.

Change the number of discussions shown within a specific forum: Select the forum where you wish to change the number of discussions shown. The existing number of discussions shown can be seen within brackets. Select then the desired new number of discussions to be shown and thereafter click on the button "Update" to change the allowed number of discussions to be displayed.


Administrate discussions

Delete a discussion: Click the check box in front of the discussion you wish to remove and then click the button "Delete ". The discussion and all contributions belonging to that discussion will be deleted.

Return to user mode by clicking the button "End admin ".

Administrate contributions

Delete a contribution: To remove a contribution, click the check box by each contribution header you want removed. Then click on the button "Delete". The contribution will then be deleted from the discussion. Important! Note the the very first contribution to a discussion, the contribution that initiated the discussion cannot be removed! To remove an initial contribution you must choose between editing the contribution or deleting the entire discussion.

Re-save an existing contribution: To edit an existing contribution, change the text as you wish and then select that contribution by clicking on the check box. Then click the button "Resave " to save the changes you made to the contribution.

Administrate templates

The template files used for the visual appearance of a conference can be changed. Note that changes should not be made in the template directory called "Original"! Instead you should create a new template directory by making a copy of "Original" and make the changes desired in this new directory.

Other. When a conference is created, a standard template directory is used called "Original". Therefore it is essential never to update the "Original" template directory!

Registering a new template directory: To create a new template directory, enter the name of the new template directory and click on button "New template directory". Note that the template directory's name must be a valid library name. For this reason use the 8.3 rule for the name of the directory. This means NO special characters should be found in the name. To be on the safe side, avoid the use of symbols or non-international characters. When the new template directory is to be created, the template files of the "Original" directory will be copied to the new directory.

Change the template directory for a conference: Select the template directory you wish to use for the conference you administrate. Click on the button "Change template directory" to switch directories. The template directory used for a conference is presented in bold style in the dialogue box.

Update template file: To update a template file of an existing template directory, first select which directory should be updated, and then select the type of template which is to be updated (imCMS-imConf supports 2 types of files (images and HTML files). When you have made this selection click on the button "Administrate".

In the subsequent dialogue box, the desired file to be uploaded can be selected. Thereafter, click on the button "Upload ". Then the template file selected will be copied to the server. Note that the existing template file will be overwritten!

Create your own template files: To create your own template pages to be used in the conference, follow the following instructions:

1. Create your own version of the templates you wish to change. The templates / images listed below must be created in every directory. Because there already exists a copy of all the necessary template files in a new directory, you do not have to create more new templates than you wish. Administrate users

Administrate users: A user who is an expert in a conference can be so denoted, that the other conference participants can easily identify and read what the expert has added. Such a user's contributions can be noted with the symbol . To allow a user to be an expert, click the button in admin mode called "Advanced administration" and then select "Administrate users". Select the user whose status you wish to change and click the button "Change". Click thereafter the check box for expert status for this particular user. Save the new status by clicking on "Save".

Administrate selfregister

Administrate selfregister: If a conference should allow users to selfregister themselves into a conference, the superadministrator could use a role and activate the property "Allow selfregister in conference" in the superadministrator tool. When this property is activated, you could add that role for the conference. That will mean that all those users who is selfregistering themselves will achive all roles for the conference.
