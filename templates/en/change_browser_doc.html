#browsers#
  Three steps to automatic switching to pages designed for specific browsers:
 1. Select and click "Add switch" for all the browsers, browser versions or computer environments for which you have already created specially tailored pages
 2. Enter the page meta-id number(s) for corresponding alternative page(s) you created for a specific environment.
 3. NB: The default "parent" page is listed automatically - which will be used for all other browsers than those specified here.
