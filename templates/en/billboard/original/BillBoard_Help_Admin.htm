Administrate billboard

Create a new billboard: Enter a name for the billboard to be created. Then click on the button "Create new billboard" to add the new billboard within the billboard. Please note the list of existing billboards.

Delete a billboard: To delete a billboard in a billboard, select that billboard you wish to remove and then click on the button "Delete". The selected billboard and all of its notices will then be deleted. Please note that a BillBoard must have at least one billboard to be able to funtion!

Change the name of an existing billboard: Select the existing billboard whose name you wish to change from the list of billboards. Enter the new name of the billboard. Click the button "Change" to let the new name replace the existing name.

Change the number of notices shown within a specific billboard: Select the billboard where you wish to change the number of notices shown. The existing number of notices shown can be seen within brackets. Select then the desired new number of notices to be shown and thereafter click on the button "Update" to change the allowed number of notices to be displayed.


Administrate notices

Delete a notice: Click the check box in front of the notice you wish to remove and then click the button "Delete ". The notice and all replies belonging to that notice will be deleted.

Return to user mode by clicking the button "End admin ".

Administrate replies

Delete a reply: To remove a reply, click the check box by each reply header you want removed. Then click on the button "Delete". The reply will then be deleted from the notice. Important! Note the the very first reply to a notice, the reply that initiated the notice cannot be removed! To remove an initial reply you must choose between editing the reply or deleting the entire notice.

Re-save an existing reply: To edit an existing reply, change the text as you wish and then select that reply by clicking on the check box. Then click the button "Resave " to save the changes you made to the reply.

Administrate templates

The template files used for the visual appearance of a billboard can be changed. Note that changes should not be made in the template directory called "Original"! Instead you should create a new template directory by making a copy of "Original" and make the changes desired in this new directory.

Other. When a billboard is created, a standard template directory is used called "Original". Therefore it is essential never to update the "Original" template directory!

Registering a new template directory: To create a new template directory, enter the name of the new template directory and click on button "New template directory". Note that the template directory's name must be a valid library name. For this reason use the 8.3 rule for the name of the directory. This means NO special characters should be found in the name. To be on the safe side, avoid the use of symbols or non-international characters. When the new template directory is to be created, the template files of the "Original" directory will be copied to the new directory.

Change the template directory for a billboard: Select the template directory you wish to use for the billboard you administrate. Click on the button "Change template directory" to switch directories. The template directory used for a billboard is presented in bold style in the dialogue box.

Update template file: To update a template file of an existing template directory, first select which directory should be updated, and then select the type of template which is to be updated imAd (billboard) supports 2 types of files (images and HTML files). When you have made this selection click on the button "Administrate".

In the subsequent dialogue box, the desired file to be uploaded can be selected. Thereafter, click on the button "Upload ". Then the template file selected will be copied to the server. Note that the existing template file will be overwritten!

Create your own template files: To create your own template pages to be used in the billboard, follow the following instructions:

1. Create your own version of the templates you wish to change. The templates / images listed below must be created in every directory. Because there already exists a copy of all the necessary template files in a new directory, you do not have to create more new templates than you wish. For example, it maybe enough for you to just change the appearance of the templates seen by users in the user mode. Please note that the names of your new files must agree with the following list.

Administrate users

Administrate users: A user who is an expert in a billboard can be so denoted, that the other BillBoard participants can easily identify and read what the expert has added. Such a user's replies can be noted with the symbol . To allow a user to be an expert, click the button in admin mode called "Advanced administration" and then select "Administrate users". Select the user whose status you wish to change and click the button "Change". Click thereafter the check box for expert status for this particular user. Save the new status by clicking on "Save".
