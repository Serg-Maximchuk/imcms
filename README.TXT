This version is tested on the following platforms and versions:

Java
This version is tested with JDK 1.4.
Should also work on newer versions of Java.

Webserver
Tested with Tomcat 4 and Tomcat 5.
Should work with any Servlet 2.3/JSP 1.2 compliant servlet engine.

Database
This version is tested on Microsoft SQL server 2000.
Should also work with Microsoft SQL Server Desktop Edition.
No other databases are currently supported.

OS
This version is tested on Windows XP, Windows 2000 and Linux.
Should also work on Windows NT.

Browsers
This version is tested with IE 6 on Windows, Firefox 0.8 on Windows, and Mozilla 1.6 on Linux.
